## Business Problem

A health clinic needs to prioritise patients with higher cardiovascular risk using simple screening information. Currently, manual specialist diagnosis reaches **55–65% precision**. Each diagnosis costs **$1,000** and the company charges a fee proportional to accuracy achieved:

| Accuracy | Fee | Revenue / patient |
|---|---|---|
| 50–55% | 5% | $50 |
| 55–60% | 10% | $100 |
| 60–65% | 15% | $150 |
| 65–70% | 20% | $200 |
| **70–75%** | **25%** | **$250** |

**A 10 pp accuracy improvement doubles revenue per patient.**

---

## Solution

A supervised binary classification model trained on 70,000 patient records from the [Kaggle Cardiovascular Disease Dataset](https://www.kaggle.com/datasets/sulianova/cardiovascular-disease-dataset).

**ML approach (Géron Ch. 2 end-to-end checklist):**
1. Frame the problem and select the performance measure
2. Get and explore the data (stratified split set aside immediately)
3. Engineer domain features (BMI, age_years, blood pressure flags)
4. Build a sklearn Pipeline (impute → scale → encode)
5. Compare models via 5-fold cross-validation (LR, Random Forest, Gradient Boosting)
6. Tune the best model with RandomizedSearchCV (30 iterations)
7. Evaluate once on the held-out test set
8. Translate ML metrics into business revenue impact
9. Serve predictions via FastAPI + Streamlit

## API Reference

| Endpoint | Method | Description |
|---|---|---|
| `/health` | GET | Health check → `{"status": "ok"}` |
| `/predict` | POST | Returns predictions + risk scores |

**Sample request:**
```bash
curl -X POST http://localhost:8000/predict \
  -H "Content-Type: application/json" \
  -d '{
    "records": [{
      "age": 19000, "gender": 2, "height": 168, "weight": 85,
      "ap_hi": 150, "ap_lo": 95, "cholesterol": 2, "gluc": 1,
      "smoke": 0, "alco": 0, "active": 1
    }]
  }'
```

**Response:**
```json
{"prediction": [1], "score": [0.73]}
```

## Top Data Insights

1. **Blood pressure is the strongest predictor** — Hypertensive patients (ap_hi ≥ 140 or ap_lo ≥ 90) have ~2× higher disease rate
2. **Age matters significantly** — Disease rate rises sharply after age 50
3. **BMI elevation is associated with risk** — Obese patients (BMI > 30) have the highest disease rate
4. **Cholesterol level has a strong signal** — "Well above normal" group shows much higher prevalence
5. **Lifestyle factors (smoke, alcohol) show weak signal** — Likely confounded by dataset collection method
